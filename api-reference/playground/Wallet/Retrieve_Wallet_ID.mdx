---
title: "Retrieve Wallet by ID"
api: "GET /wallets/get/wallet/{walletId}"
description: "The endpoint retrieves the details of a specific wallet using its unique wallet ID."
---

## Path Parameters

<ParamField body="id" type="string" required>
	Unique identifier for the wallet. **Example:**
	"3fa85f64-5717-4562-b3fc-2c963f66afa6"
</ParamField>
