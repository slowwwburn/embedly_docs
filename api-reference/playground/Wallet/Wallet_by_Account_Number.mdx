---
title: "Wallet by Account Number"
api: "GET /wallets/get/wallet/account/{accountNumber}"
description: "The endpoint retrieves the details of a specific wallet using its associated account number.."
---

## Path Parameters

<ParamField path="accountNumber" type="string" required>
	Unique identifier for the account
</ParamField>

  }
}
