---
title: "Update Corporate Customer"
api: "PUT /corporate/customers/{customerId}/directors/{directorId}"
description: "This endpoint enables you to create a corporate customer profile"
---

---

## Request Parameters

<ParamField path="customerId" type="string" required>
	The unique identifier of a corporate customer. **Example:** `123456`
</ParamField>

<ParamField path="directorId" type="string" required>
	The unique identifier mapped to a corporate customer's director. **Example:**
	`01234567-0001`
</ParamField>

```json
    }
    "errors": [],
}
```

</ResponseExample>
